1.导入项目

file->new->project from existed source

选择eclipse项目，其它步骤默认即可

2. 配置项目

打开右上角的project setting

2.1 project

一般不会错，注意一下jdk的版本与开发项目的版本是否一致即可

2.2 modules

​ + ,添加web,文件选择WebContent->WEB-INF->web.xml。同时Web Resource中选中WebContent的路径-注意不能是web

2.3 libraries

+,添加java,将lib下的jar包和引用的jar包引入

2.4 artifacts

+,添加web,选中在2.2模块中配置好的Module

2.5 artifcats

+,添加Web Applicaton:exploaded或者Web Application:archive中任选一个，module从2.2中配置中的选择
